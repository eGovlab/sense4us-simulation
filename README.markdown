# Sense4us Simulation

## Compiling
1. ```grunt```
   To build the tool.
2. ```grunt watch```
   To build the tool on file change.

## If you want to run a server which serves the tool
1. ```node bin/init.js```

## To generate the code documentation
1. ```grunt jsdoc```
   Default target is docs.
