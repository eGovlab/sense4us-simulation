sense4us.events Class

Defined in: js\events.js:7

This is an event handler, meaning that we can bind a callback to an event. This will then be called when we run the trigger-function. This is basically a static function, so it's accessible from any place using sense4us.

Constructor

sense4us.events

()

Defined in js\events.js:7

Example:

We want the function eatPizza(pizza) to be called on the event "pizzaDelivered":

sense4us.events.bind("pizzaDelivered", eatPizza);

To trigger the event:

sense4us.events.trigger("pizzaDelivered", pizzaObject);
